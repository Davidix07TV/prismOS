# prismOS Release Plan v0.1

## Objective
Publish the first bootable prototype focused on prismOS Work.

## Scope
- Ubuntu LTS base
- KDE Plasma desktop
- Basic branding (name and wallpaper)
- Work profile default apps and settings

## Out of Scope
- Full EDU policy engine
- Advanced Games tuning
- Complete Soft optimizations
- Custom installer UI

## Milestones

### M1 - Base Setup
- Confirm base distro version
- Define core package list
- Create initial build instructions

### M2 - Work Prototype
- Apply Work defaults
- Add productivity app bundle
- Disable non-essential startup items

### M3 - Internal Test
- Boot test on virtual machine
- Validate performance and stability
- Collect known issues list

### M4 - v0.1 Publish
- Tag repository with v0.1-pre
- Publish notes and installation steps
