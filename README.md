# prismOS 💎✨

prismOS is a Linux-based operating system project inspired by a clean, elegant desktop experience, with multiple editions for different needs.

## Authors
- Davidix
- Emaioyoce

## Editions

### prismOS EDU
Education-focused edition.
- School-controlled app policies
- Blocking for non-educational apps/games
- Classroom-friendly defaults
- Safe browsing and restricted mode options

### prismOS Work
Productivity-focused edition.
- Lightweight, stable desktop workflow
- Office and collaboration first
- Optimized multitasking and responsiveness
- Minimal distractions by default

### prismOS Games
Gaming-focused edition.
- Performance tuning profile for higher FPS
- Reduced background services while gaming
- Game launcher integration
- Low-latency configuration presets

### prismOS Soft
Legacy hardware edition.
- Reduced animations and visual effects
- Lower RAM/CPU footprint
- Fast boot on older machines
- Essential features only

## Base Vision
- Linux-based distribution with modular profiles
- Unified design language across all editions
- Shared core packages + edition-specific overlays
