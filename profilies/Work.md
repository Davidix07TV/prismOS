# prismOS Work

## 1) Purpose
Productivity-first environment for professional and office usage.

## 2) Enabled Features
- Focus mode defaults
- Fast window/task switching
- Cloud sync integration options
- Notification minimization

## 3) Disabled Features
- Non-essential startup services
- Distracting default widgets
- Optional block for gaming launchers

## 4) Default Apps
- Office suite
- Calendar and mail client
- Notes/task manager
- Video meeting client
- Password manager (optional)

## 5) Performance Policy
- Balanced-to-performance profile
- Optimized multitasking
- Controlled background updates

## 6) Admin Controls
- Default app bundle selection
- Security policy presets
- Device encryption policy
- Update window scheduling
