# prismOS Games

## 1) Purpose
Gaming-focused environment optimized for FPS, latency, and smoothness.

## 2) Enabled Features
- Performance mode preset
- Game session optimizer
- Full-screen priority behavior
- Optional game launcher hub

## 3) Disabled Features
- Heavy background indexing during gameplay
- Non-critical startup services
- Aggressive background sync jobs

## 4) Default Apps
- Graphics driver tools
- Game launcher support tools
- Performance monitor
- Voice chat client (optional)

## 5) Performance Policy
- High-performance CPU governor
- Reduced background tasks
- Lower input latency settings
- Fast network profile tuning

## 6) Admin Controls
- Toggle aggressive performance mode
- Select launcher ecosystem
- Configure thermal/performance limits
- Enable/disable overlays
