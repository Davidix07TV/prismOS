# prismOS EDU

## 1) Purpose
School-focused environment with controlled access and reduced distractions.

## 2) Enabled Features
- Restricted user mode
- Application whitelist policy
- Web filtering option
- Classroom preset support

## 3) Disabled Features
- Non-approved games
- Non-approved entertainment apps
- Optional block on app stores (school choice)

## 4) Default Apps
- Browser (education-safe config)
- Office suite
- PDF reader
- Learning platform clients (optional)

## 5) Performance Policy
- Balanced power profile
- Low background activity
- Fast session startup

## 6) Admin Controls
- Add/remove apps from whitelist
- Choose blocked categories
- Configure web filtering level
- Lock system settings
