# prismOS Soft

## 1) Purpose
Lightweight environment for older or low-spec hardware.

## 2) Enabled Features
- Lightweight desktop behavior
- Reduced visual effects
- Low-memory defaults
- Essential-services-only startup

## 3) Disabled Features
- Heavy animations and blur effects
- Non-essential background daemons
- Optional disable of auto-indexing

## 4) Default Apps
- Lightweight browser
- Lightweight text editor
- Basic office tools
- Simple media player

## 5) Performance Policy
- Power-saving balanced profile
- RAM-conscious service limits
- Minimal startup footprint
- Fast boot preference

## 6) Admin Controls
- Visual effects level
- Startup service selection
- Memory usage caps for optional services
- Simplified update mode
