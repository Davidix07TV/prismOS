# Profile Configuration Format

Each profile file should define:

## 1) Purpose
Short description of the profile goal.

## 2) Enabled Features
List of features enabled by default.

## 3) Disabled Features
List of features disabled by default.

## 4) Default Apps
Preinstalled apps for that profile.

## 5) Performance Policy
CPU/GPU/background-services behavior.

## 6) Admin Controls
What can be configured by admin/school/company.
